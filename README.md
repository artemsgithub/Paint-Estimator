# Paint Estimator

A mobile-friendly React web app for self-employed painters to estimate jobs on-site.

## Phase 1 — Estimate Builder

- Add labor tasks (description, hours, rate) with live row totals
- Add materials (item, qty, unit cost, markup %) with live row totals
- Adjustable contingency percentage
- Sticky summary bar with grand total always visible
- Mobile-first, single-column layout with numeric keyboard support

## Tech Stack

- React 19 + Vite
- Vanilla CSS (no dependencies)
- Deploys to Vercel with zero config

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output goes to `dist/` — Vercel auto-detects Vite projects.
